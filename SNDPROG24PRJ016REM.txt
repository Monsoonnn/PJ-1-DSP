Để có thể chạy được các file cpp, cần cài đặt gnuplot.info
http://www.gnuplot.info/
Đồng thời thực hiện các bước sau
1. Mo Control Panel trên Windows và di den System and Security > System > Advanced system settings.
2. Trong hop thoai System Properties, chon tab Advanced, và nhap vào nút Environment Variables.
3. Trong phan System Variables, tìm bien môi tru?ng có ten PATH và chon Edit.
4. Them duung dan den thu muc cai dat Gnuplot vào cuoi bien PATH (C:\Program Files\gnuplot\bin).
5. Nhap OK de luu thay doi
